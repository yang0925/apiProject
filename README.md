# apiProject
  <h2>🔎 API 통신을 이용한 사진 검색 앱 만들기</h2>
  
  <h2>❗ 기능</h2>
   
  ✔ 지정된 글자수까지 검색어 입력
  
  ✔ grid recyclerview를 이용해 결과 보여주기
  
  ✔ 돋보기 모양 클릭 후 새로운 검색어 입력 가능
  
  ✔ 새로운 검색어에 따른 새로운 결과 보여주기
   
  <h2>❕ 기술</h2>
  
  ✔ Retrofit + okhttp
  
  ✔ Shared Prefrence
  
  ✔ RxJava
  
  ✔ coroutine
  

  <h2>🛠 사용툴/언어</h2>

  ◾ Kotlin

  ◾ Android Studio
 
 
  <h2>🔁 개발 순서</h2>
  
  ◾ material theme, extension(UI 작업)
  
  ◾ Retrofit
  
  ◾ grid recyclerview
  
  ◾ searchView
  
  ◾ shared preference manager
  
  ◾ latest searchterm feature(최근 검색어 저장)
  
  ◾ RxJava
  
  ◾ coroutine flow
